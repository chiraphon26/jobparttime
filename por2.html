VPN

VPN คืออะไร

VPN : Virtual Private Network เป็นฟังก์ชันทางเน็ตเวิร์คที่ใช้สำหรับทำให้การรับส่งข้อมูลบน Network มีความปลอดภัยมากยิ่งขึ้น โดยความปลอดภัยที่มากยิ่งขึ้นนั้นมากจากการที่ VPN เป็นการสร้าง Gateway เฉพาะในการรับส่งข้อมูล มีการ login password ในการเข้าถึงทุกคร้ัง หรือเรียกได้ว่า VPN เป็นสร้างเน็ตเวิร์คส่วนตัวขึ้นมาใช้งานนั่นเอง

VPN ทำอะไรได้บ้าง

- เเม้ว่า การติดต่อสื่อสารของเราได้ติดต่อออกไปนอก LAN การติดต่อสื่อสารก็ยังมีความปลอดภัย

- VPN network สามารถทำได้ทุกที่

- มีการเข้ารหัสข้อมูลก่อนส่งทุกครั้ง

- เราสามารถที่จะเข้าถึงเว็บไซต์ที่โดนบล๊อคการบริการเช่น เราอยู่ในไทยเราโดยบล้อคห้ามเข้า Facebook เราสามารถเปลี่ยนภูมิภาคของ VPN server ไปใช้ของอเมริกาเเละสามารถเล่นตามปกติได้

- สามารถปกปิดตัวตนได้ เนื่องจากว่า เราสามารถปลอมเป็นผู้ใช้จาก ประเทศ A ทั้งๆที่เราอยู่ประเทศ B ได้
